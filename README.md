# Wasim Akram Sheriff - Personal Portfolio 🚀

A responsive, interactive personal portfolio website showcasing my skills, projects, and professional experience in Software Engineering, AI, and Cloud Computing. 

**🔗 [View Live Website Here](https://wasim-akram-sheriff1012.netlify.app/)**

## 🌟 Features
* **Modern UI/UX:** Clean, dark-themed design with smooth scrolling and engaging hover animations.
* **Fully Responsive:** Optimized for both desktop and mobile viewing.
* **Interactive Elements:** Features a dynamic typing effect, scroll-reveal animations, and a custom loading screen.
* **Project Showcase:** Detailed cards highlighting academic and real-world projects, including my AI Smart Traffic System and Home Automation concepts.

## 🛠️ Built With
* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom styling, CSS grid/flexbox, animations, and responsive media queries.
* **JavaScript (Vanilla):** DOM manipulation, scroll event handling, and typing effects.
* **Font Awesome:** Iconography for contact links and timeline items.

## 💻 How to Run Locally
If you want to run this project on your local machine:
1. Download the code from GitHub as a ZIP file and extract it.
2. Open `index.html` in any web browser. (No build tools or server required!)
